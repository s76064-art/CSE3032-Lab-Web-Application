# CSE3032-Lab-Web-Application

# Lab2 

## Description
This lab focuses on setting up the complete development environment for Java web applications. It covers installing essential software, configuring tools, and creating basic Java Servlet and JSP programs.

## What I Achieved
- Installed and verified **JDK**, **XAMPP**, **Apache Tomcat**, and **NetBeans IDE**.
- Learned how to **configure Tomcat inside NetBeans** for running web applications.
- Practiced **changing the default MySQL root password** and managing services using XAMPP.
- Learned how to **write and run a simple Java Servlet**.
- Learned how to **create and execute a basic JSP program**.
- Explored using **Java reference types and wrapper classes in JSP**.
- Learned how Servlets handle **data sharing** between web pages (e.g., login → account page).
- Created a **MySQL database and tables** for use in web applications.
- Set up a project environment to enable **database connectivity** using MySQL Connector/J.
- Implemented **CRUD operations (Create, Read, Update, Delete)** using Servlets.
- Applied **DAO (Data Access Object)** and **JavaBeans** to structure database-driven web applications effectively.
